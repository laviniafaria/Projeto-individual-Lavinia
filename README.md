# Projeto Individual: Baralho Cigano 

Este repositório foi criado para armazenar todos os conteúdos e desenvolvimento do meu projeto individual, focado no **Baralho Cigano**.

O objetivo central é incentivar o **conhecimento** sobre as cartas e promover seu uso como uma valiosa **ferramenta de desenvolvimento pessoal e espiritual**.

---

## Baralho Cigano

### Sobre o Projeto

Este projeto nasceu da minha paixão pelo Baralho Cigano, inspirada pela minha **trajetória espiritual e experiências mediúnicas**.

Com o tempo, desenvolvi uma forte afinidade com as leituras e reconheci o quanto essa ferramenta pode auxiliar no **autoconhecimento** e na **conexão espiritual**.

### Objetivo

Promover o conhecimento e a prática do Baralho Cigano, tornando-o acessível para quem deseja:

* Compreender seus **significados e símbolos**.
* Aproveitar sua capacidade de **orientar caminhos e reflexões internas**.
